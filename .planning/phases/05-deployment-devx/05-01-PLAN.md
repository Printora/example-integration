  <name>Task 3: Add deployment scripts to package.json</name>
  <files>package.json</files>
  <action>
Add the following to package.json:

1. Add "engines" field at the root level (not inside scripts):
```json
"engines": {
  "node": ">=20.9.0"
}
```

2. Add a "deploy:vercel" script in the scripts section:
```json
"deploy:vercel": "vercel --prod"
```

Rationale: The engines field documents the minimum Node.js version required (Next.js 16 requirement). The deploy:vercel script provides a convenient command for production deployment.

Note: Vercel automatically uses Node.js 20+ for Next.js 16 projects, but the engines field documents the requirement for local development and alternative hosting.
  </action>
  <verify>node -e "const pkg=require('./package.json'); console.assert(pkg.engines?.node?.includes('20'), 'engines.node missing'); console.assert(pkg.scripts['deploy:vercel'], 'deploy:vercel script missing')"</verify>
