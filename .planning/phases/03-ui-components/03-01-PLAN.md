  <name>Install shadcn/ui dependencies and create utility function</name>
  <files>package.json, lib/utils.ts</files>
  <action>
    1. Install required utility packages:
       ```bash
       npm install clsx tailwind-merge class-variance-authority lucide-react
       ```

    2. Create lib/utils.ts with the cn utility function:
       ```typescript
       import { clsx, type ClassValue } from "clsx"
       import { twMerge } from "tailwind-merge"

       export function cn(...inputs: ClassValue[]) {
         return twMerge(clsx(inputs))
       }
       ```

    Note: This follows the standard shadcn/ui pattern for conditional className merging. The cn function combines clsx (conditional classes) with tailwind-merge (removes conflicting Tailwind classes).
  </action>
  <verify>
    1. Check packages are in package.json: `grep -E "clsx|tailwind-merge|class-variance-authority|lucide-react" package.json`
    2. Verify lib/utils.ts exists and exports cn: `cat lib/utils.ts | grep "export function cn"`
  </verify>
  <done>
    - clsx, tailwind-merge, class-variance-authority, lucide-react are in package.json dependencies
    - lib/utils.ts exists with cn function that combines clsx and twMerge
  </done>
